#  ローグライクビンゴ

# 概要
・「列減少」「列増加」「素数禁止」「完全数禁止」などのパークがある
・早く上がるのが目的ではなく、ハイスコアを目指すゲームで、ビンゴしても続く
・一定回数数字を回したら終了。ビンゴするごとに回せる残り回数が増加。
・残り回数が0になるかすべてのマスが開くと終了で、スコアとそれに応じたランクを結果として表示。Xに共有できるようにする
・穴が空いたときに数字に応じたスコアを獲得。
・ビンゴしたときに、成立した列の数と使った数字に応じて大きなスコアを獲得
・5回回すごとに、ランダムでパークが3つ提示され、うち1つを選択できる

# アイデア
・出る数字が整数である必要すらない。たとえば、負の数が出るようになるパークがあってもいい

# パークの例
・列減少；現在5x5なら、4x4になる。出る数字の範囲も拡大
・列増加；現在5x5なら、6x6になる。出る数字の範囲も拡大
・素数禁止；素数が出なくなる
・完全数禁止；完全数が出なくなる
・負数開放；負数が出るようになる。負数のマスは列増加で拡張されたマスで出現するかもしれない
・絶対値判定：数字の一致ではなく絶対値で判定する
